HTML INTERVIEW QUESTIONS

  1. What are HTML tags?
    A.HTML tags are keywords enclosed in angle brackets (< >) that define the structure and content of a webpage.

  2. What are HTML attributes?
    A.Attributes provide additional information about HTML elements and are placed within the opening tag.

  3. What is the difference between an HTML tag and an HTML element?
    An HTML tag is the code enclosed in angle brackets, while an HTML element consists of the opening tag, closing tag (if needed), and the content in between.
